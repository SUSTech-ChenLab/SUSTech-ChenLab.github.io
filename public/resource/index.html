Here is some useful resource generated by the lab about genomics, especially single cell genomics. Drop us an email if you have any questions.

1. Are you bombarded by all sorts of single cell methods and unable to figure out their difference?

  • Check the scg_lib_structs GitHub repository on the library structures of popular single cell methods.

2. Do you want a uniform pipeline for the preprocessing of your single cell data?

  • Check the Single Cell Genomics Library Preprocessing Pipelines ReadTheDoc documentation.

3. A Snakemake pipeline to preprocess the plate-based single cell ATAC-seq data

  • Check the scATAC_snakemake GitHub repository.
